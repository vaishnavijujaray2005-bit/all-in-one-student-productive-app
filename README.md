# Student Productivity App 📚🚀

## Overview

Student Productivity App is a mobile UI/UX solution designed to help engineering students efficiently manage their academic activities from a single platform. The application provides a centralized dashboard where students can track their attendance, assignments, internal marks, class schedules, and important notifications.

The goal of this project is to reduce the difficulty of switching between multiple sources of academic information and create a seamless, organized, and student-friendly experience.

---

## Problem Statement

College students often struggle to keep track of attendance percentages, assignment deadlines, internal assessment marks, and daily class schedules. Academic information is usually scattered across different platforms, resulting in missed deadlines and poor time management.

This project aims to solve these challenges by designing an all-in-one digital companion that helps students stay informed and organized.

---

## Target Users

* 2nd, 3rd, and 4th year engineering students
* Students managing multiple subjects and assignments
* Students who want a simple way to monitor their academic progress

---

## Key Features

###  Academic Dashboard

* Overview of attendance percentage
* Upcoming assignments
* Internal marks summary
* Important academic updates

###  Timetable Management

* Daily and weekly class schedule
* Subject and faculty details
* Easy access to upcoming classes

###  Assignment Tracker

* View assignments subject-wise
* Track submission deadlines
* Check assignment priority and completion status

###  Attendance Tracker

* Monitor attendance percentage for each subject
* Identify subjects with low attendance

###  Internal Marks Tracker

* View marks obtained in internal examinations
* Track academic performance throughout the semester

###  Notifications

* Receive reminders for assignments, exams, and academic announcements

---

## Design Process

### 1. User Research

Understanding the challenges faced by engineering students regarding academic management and time organization.

### 2. Information Architecture

Organizing academic features into a clear navigation system for quick access.

### 3. Wireframing

Creating low-fidelity layouts to define the user flow and screen structure.

### 4. UI Design

Developing high-fidelity mobile interfaces with a modern and clean visual style.

### 5. Prototyping

Building interactive prototypes to simulate real user interactions and navigation.
